# 🛡️ NEKUTA LMRA Digital

**Analyse des Risques de Dernière Minute — VCA★★**

Application mobile-first pour la réalisation des LMRA sur chantier.

## Démarrage rapide

```bash
npm install
npm run dev
```

## Déploiement

```bash
npm run build
```

Le dossier `dist/` contient les fichiers à déployer sur Vercel, Netlify ou tout autre hébergeur statique.
